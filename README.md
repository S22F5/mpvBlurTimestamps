# mpvBlurTimestamps

### Usage
1. move playLinux.sh or playWindows.ps1 into the same folder as your video.
2. change the wildcard ***.mkv** in the script *(optional)*.
3. create a timestamps.txt file in the same folder (examples provided).
4. run playLinux.sh or playWindows.ps1.

### timestamps.txt Format
- Line 1: **Name** of video for which timestamps are for.
- Line 2: **Playtime** in seconds eg. 5623.720s.
- Line 3-X: **Timestamps** in HH:MM:SS HH:MM:SS.
